# KBO_Project 사용자 가이드 (User Guide)

## 서비스 소개

** KBO 야구 팬 성향분석 및 구단별 관중수 증가 요인분석

> **🚀 Live Demo: [https://kboproject1.streamlit.app/](https://kboproject1.streamlit.app/)**

---

> [!IMPORTANT]
> **2025년 데이터 기준 안내**
> 본 프로젝트는 **2025년 KBO 리그 일정 및 데이터**를 기반으로 제작되었습니다.
> 현재(2026년 이후) 실행 시, 미래 데이터가 없어 경기 일정이나 추천 결과가 나오지 않을 수 있으나,
> **자동으로 가장 최근 데이터(2025년 8월 31일 기준)를 표시하도록 보정 로직이 적용되어 있습니다.**
> (예: "오늘의 경기"가 없을 경우, 2025년의 마지막 경기 일정을 대신 보여줍니다.)

## 주요 기능

### 승률 - 관중수 상관관계

- 기간, 구단 선택에 따른 승률, 관중수 상관관계 인사이트 제공
- 승률 vs 평균관중수 산점도 차트
- 연도별 승률 & 관중수 트렌드 차트
- 팀별 승률 및 관중수 순위 차트
- 분석 결과 및 해설 제공

### SNS 팔로워 vs 관중수 + 구단 나이 상관 관계

- 기간, 구단 선택에 따른 SNS 팔로워 vs 총관중수 산점도, 구단 나이 상관관계 인사이트 제공
- SNS 팔로워 vs 총관중수 산점도 차트
- 구단 나이 vs 총관중수 산점도 차트
- 연도별 SNS 팔로워 & 관중수 트렌드 차트
- 분석 결과 및 해설 제공

### 야구 경기 일정

- 야구 경기 일정 제공

### 야구 직관 팬 성향 분석

- 다양한 선택지에 따른 팬성향 결과지 도출
- 유저 선택에 따른 구장, 좌석, 맛집 추천

- 유저 선택에 따른 구장, 좌석, 맛집 추천

### ✅ 최근 수정 사항 (Updates)

**1. 오류 수정 (Bug Fixes)**
- **API 키 로드 문제 해결**: `.env` 파일 로드 로직 추가로 배포 환경 오류 해결
- **경기 일정 미표시 해결**: 2026년에 실행 시 2025년 데이터가 없어 비어 보이던 문제 -> **최근 데이터(2025-08-31) 자동 표시** 기능 추가
- **구장 추천 오류 해결**: 미래 경기가 없을 때 발생하던 `TypeError` 수정 및 **랜덤 추천 로직** 추가 (다양한 팀 추천)

**2. UI 개선 (UI Improvements)**
- **중앙 정렬 적용**: 시작 화면, 질문 화면, 결과 화면의 모든 텍스트와 버튼을 중앙 정렬하여 디자인 통일
- **반응형 디자인**: 맛집 추천 박스의 텍스트가 길어질 경우 **자동으로 줄바꿈 및 높이 조절**되도록 CSS 개선
- **Mac 폰트 호환성**: MacOS 환경에서 한글 폰트 깨짐 현상 해결 (`AppleGothic` 적용)

## 협업 방식

- **포크(Fork) 기반 협업**: 메인 리포지토리를 포크하여 개발합니다.
- 개발 완료 후, **Pull Request(PR)** 를 생성합니다.
- PR에 대해 **코드 리뷰**가 진행됩니다.
- 리뷰 승인 후, **Main Repository에 Merge** 됩니다.

---

## 커밋 컨벤션

커밋 메시지는 아래 컨벤션을 따릅니다:

- feat: 유저 로그인 기능 추가
- fix: 로그인 시 비밀번호 오류 해결
- docs: 환경설정 가이드 추가
- refactor(api): API 요청 방식 개선
- chore: 패키지 정리 및 의존성 업데이트

## 프로젝트 기본 폴더 구조

```
.
├── README.md
├── app
│ ├── \__init_.py
│ ├── components
│ └── *_screen.py
├── apis : 리소스 폴더
├── aseets : 리소스 폴더
├── models : 데이터 모델링
├── data : 데이터 저장소
├── requirements.txt : 사용 라이브러리 명시
├── scraper : 크롤링 로직
├── features : 기능
├── main.py : streamlit 진입점
└── libs : util 함수

```

## 네이밍 규칙

- 파일 네임, 변수, 함수 : snake_case
- 클래스: PascalCase
- 상수: ALL_CAPS

---

## 커밋 메세지 prefix

- feature/ : 새로운 기능 설명
- fix/ : 버그 수정
- refactor/ : 코드 리팩토링 (기능 변화 없음)
- chore/ : 환경 설정, 패키지, 문서 등
- docs/ : 문서 관련 수정
